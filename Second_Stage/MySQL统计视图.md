# MySQL统计视图

## 1.什么是统计视图

统计视图是一种虚拟表，不存储实际数据，而是存储了一个SQL查询语句，当查询这个统计视图时，MySQL会执行这个语句并返回结果

核心特点：封装复杂的统计逻辑，简化查询；统一数据口径，避免查出不同结果；保护原始数据，只暴露出需要统计的信息；像普通表一样进行查询和过滤

## 2.基本语法

```sql
create [or replace] view 视图名 [(列名1，列名2,...)]
as 
select 聚合函数(列名),分组列1，分组列2,... from 表名 [where 条件] group by 分组列1，分组列2,... 
[with [cascaded | local] check option]
```

常用聚合函数：

- `COUNT(*)`：统计行数
- `COUNT(DISTINCT 列名)`：统计不重复值的数量
- `SUM(列名)`：求和
- `AVG(列名)`：平均值
- `MAX(列名)`：最大值
- `MIN(列名)`：最小值
